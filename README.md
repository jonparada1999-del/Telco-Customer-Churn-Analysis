# Telco-Customer-Churn-Analysis
https://public.tableau.com/app/profile/jonathan.parada/viz/ExecutiveDashboardTelcoCustomerRetentionAnalysis/Dashboard1?publish=yes

Goal: Predicted customer churn with 78.5% accuracy using a Random Forest model in Python.

Insight: Identified that Month-to-month contract holders represent the highest churn risk group.

Result: Visualized an overall 26.5% churn rate in an executive Tableau dashboard to drive retention strategy.
